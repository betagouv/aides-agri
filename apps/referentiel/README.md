# referentiel

## Dépendances externes

* Le référentiel du découpage administratif : ce service public numérique (https://geo.api.gouv.fr/decoupage-administratif) expose des jeux de données complets concernant les régions, collectivités d’outre-mer, départements, EPCIs, et communes.

## Dépendances logicielles

* [django-enum](https://pypi.org/project/django-enum/) : gestion puissante de types énumérations dans les modèles ;
* [djangorestframework](https://pypi.org/project/djangorestframework/) : framework-dans-le-framework pour gérer des API JSON/HTTP ;
